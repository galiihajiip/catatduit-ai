# CatatDuit AI - Web Dashboard

Next.js web app untuk CatatDuit AI personal finance manager.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Buka [http://localhost:3000](http://localhost:3000)

## Deploy ke Vercel

### Option 1: Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2: GitHub Integration
1. Push ke GitHub
2. Buka [vercel.com](https://vercel.com)
3. Import repository
4. Deploy otomatis

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Recharts
- Zustand
- Lucide Icons

src/
